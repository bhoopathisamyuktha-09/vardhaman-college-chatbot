# Vardhaman College Chatbot

A responsive chatbot web application designed to provide quick and easy access to information about Vardhaman College of Engineering.

## Features
- Interactive chatbot UI
- Quick-access buttons for college-related information
- Responsive design for desktop and mobile devices
- Clean and modern user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## AI Assistance
This project was developed with the assistance of **Cursor AI** for code suggestions and debugging.
The overall structure, design, and implementation were done by me.

## How to Run
1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Start interacting with the chatbot

## Future Improvements
- Backend integration
- Real-time responses using APIs
- Improved NLP-based chatbot logic
